# 项目章程（Project Charter）
项目名称：校园活动报名与签到管理系统

## 1. 项目背景
为满足校园活动高效管理需求，实现活动发布、报名、审核、签到与统计一体化。

## 2. 项目目标
- 完成活动管理、报名、签到、统计功能的项目管理设计
- 输出规范的WBS、进度计划、配置管理方案
- 通过Git协作完成版本控制与冲突解决

## 3. 项目范围
包含：活动发布、报名、审核、签到、统计导出
不包含：支付、统一身份认证、高并发优化、App开发
